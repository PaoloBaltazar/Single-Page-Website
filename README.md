# Real Estate Website

A modern, responsive landing page single page website

## Tools & Technologies Used

### Core

- **HTML5** — Semantic markup and page structure
- **CSS3** — Custom styling, animations, and responsive design
- **JavaScript (Vanilla)** — Scroll animations, navigation behavior, parallax effects, and counter animations

### Fonts

- **Google Fonts**
  - [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) — Display/heading font
  - [Outfit](https://fonts.google.com/specimen/Outfit) — Body/UI font

### Integrations

- **Google Maps Embed API** — Interactive office location map

## Project Structure

```
├── index.html              # Main HTML page
├── css/
│   └── styles.css          # All styles, animations, and responsive breakpoints
├── js/
│   └── main.js             # Loader, navigation, scroll animations, parallax, counters
├── images/
│   ├── about/              # About section headshot
│   ├── hero section/       # Hero background image
│   ├── gallery/            # Photo gallery images (Gallery 1–5)
│   ├── logos/              # Dark and Light brand logos
│   ├── marquee logos/      # Affiliation logos (Ridge Realty, Realtor, Equal Housing, Pahrump Valley)
│   └── services/           # Service card images
└── README.md
```

## Features

- Responsive design across desktop, tablet, and mobile
- Smooth scroll navigation with animated navbar
- Hero section with parallax zoom effect
- Scrolling logo marquee banner
- Service cards with hover animations
- Photo gallery with overlay effects
- Animated statistics counters
- Testimonial section
- Google Maps embed for office location
- Contact form with styled inputs
- Full footer with navigation, office details, social links, and affiliation logos
